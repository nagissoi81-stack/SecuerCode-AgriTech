## 🚀 حول المشروع | About The Project
**AgriSmartTech** هو نظام إداري متطور يهدف إلى دعم المهندسين الزراعيين والمزارعين في تحسين جودة المحاصيل (مثل البرسيم والطماطم). يركز النظام على إدارة التسميد، مكافحة الآفات، ومواجهة تحديات الإجهاد الحراري باستخدام تقنيات الويب الحديثة.

**AgriSmartTech** is an advanced management system designed to support agricultural engineers and farmers. It focuses on optimizing crop quality (e.g., Alfalfa and Tomatoes) by managing fertilization, pest control, and heat stress through modern web technologies.

## ✨ المميزات الرئيسية | Key Features
* **Smart Dashboard:** واجهة تفاعلية لمتابعة حالة المحاصيل.
* **Soil Nutrition Management:** أدوات لحساب احتياجات التربة من الأسمدة.
* **Pest Control Guide:** دليل شامل لمكافحة الآفات (مثل حافرة أنفاق الطماطم).
* **Bilingual Support:** دعم كامل للغتين العربية (RTL) والإنجليزية.
* **Security First:** تصميم آمن برؤية خبير في الأمن السيبراني.
